FAI-Seminar: Logistics

Hi, 感谢大家来到 FAI-Seminar！我们期待能有一个高质量讲座的分享，因此 FAI-Seminar 就这样诞生了。 在每次 talk 中，我们会邀请同龄人中最杰出的一批学者来进行学术分享。 我们希望这里是一个好玩、且有意义的地方。 所以... 欢迎来玩！（请关注公众号【人工智能基础研究】发送【FAI】加入微信群）

FAI-Seminar: Regular Talk / 常规分享


在每次talk中，会有一位speaker来进行学术分享。以下为研讨班的基本安排 (update: 2025/2/12)
  • Talk 时间为每周五上午10:00-11:00，如有变动会提前一天以上告知
  • 在机器学习三大会（ICML、NeurIPS、ICLR）投稿前三周、投稿后一周、开会当周；以及春节前后两周休息
  • Talk 时长为 45-60 分钟，QA环节时间不限
  • Talk 信息会提前一周发布在公众号与本网站上、提前一天（每周四）发布在群里（包括讲座信息和腾讯会议链接）
  • 在 Talk 中，speaker 将不时发布QA环节，大家可以通过腾讯会议链接开麦发言
  • 录屏视频会尽快（一般三天内）发布到B站账号@FAI-Seminar中

    • FAI-Seminar: Special Issues / 特别活动


    为了让研讨班更有活力，我们会不定期举办一些特别活动，包括讲座、采访、圆桌等形式。相关信息会提前发布在本网站中。
  • 每轮 seminar 中，我们会邀请至少一位老师进行 special talk，时间会提前发布（请关注群公告）
  • 每轮 seminar 可能还会组织一个讲者圆桌，邀请本轮中出现的 speaker 一起聊天
  • 我们还可能邀请一些业界相关人士，希望跨界的碰撞会擦出更多有趣的火花

    • FAI-Seminar: Speaker Invitation / 讲者招募


    我们希望带给大家高质量的分享，因此设计了如下讲者招募机制。如果您有更好的想法，请随时联系我们！
  • 通过自荐渠道（右侧讲者报名）提交申请的朋友，经 FAI 社区评估后确认是否进入候选池；
  • 所有 FAI 贡献者（给过talk的speaker）均有权提名讲者，被提名者将自动进入候选池；
  • 每月末，全体 FAI 贡献者进行投票，依据票数高低确定次月的讲者邀请名单；
  • 所有 FAI 贡献者无需参与投票环节，即可直接申请在次月进行分享。
