FAI-Seminar: Logistics

Hi, 感谢大家来到 FAI-Seminar！我们期待能有一个高质量讲座的分享，因此 FAI-Seminar 就这样诞生了。 在每次 talk 中，我们会邀请同龄人中最杰出的一批来进行学术分享。 我们希望这里是一个好玩、且有意义的地方。 所以... 欢迎来玩！（请联系 WeChat: adoutengjiaye 备注【FAI+姓名+学校】加入微信群）

FAI-Seminar: Regular Talk / 常规分享


在每次talk中，会有一位speaker来进行学术分享。以下为研讨班的基本安排 (update: 2023/5/20)
  • 每年会有三轮 regular talk，以机器学习三大会（ICML、NeurIPS、ICLR）的投稿时间作为分割
  • 每轮会包含8-12个 talk
  • Talk 时间为每周五上午10:00-11:00，如有变动会提前一天以上告知
  • Talk 时长为 45-60 分钟，QA环节时间不限
  • Talk 信息会提前一周发布在本网站上、提前一天（每周四）发布在群里（包括讲座信息和腾讯会议链接）
  • 在 Talk 中，speaker 将不时发布QA环节，大家可以通过腾讯会议链接开麦发言
  • 录屏视频会尽快发布到B站账号@滕佳烨中

    • FAI-Seminar: Special Issues / 特别活动


    为了让研讨班更有活力，我们会不定期举办一些特别活动，包括讲座、采访、圆桌等形式。相关信息会提前发布在本网站中。
  • 每轮 seminar 中，我们会邀请至少一位老师进行 special talk，时间会提前发布（请关注群公告）
  • 每轮 seminar 可能还会组织一个讲者圆桌，邀请本轮中出现的 speaker 一起聊天
  • 我们还可能邀请一些业界相关人士，希望跨界的碰撞会擦出更多有趣的火花

    • FAI-Seminar: Speaker Invitation / 讲者招募


    我们希望带给大家高质量的分享，因此设计了如下讲者招募机制。如果您有更好的想法，请随时联系我们！
  • 所有给过 talk 的 speaker 会被认证为 seminar contributor
  • 所有 seminar contributor 均有权提名推荐 speaker
  • 在每一轮 regular talk 前，所有 seminar contributor 会一起对提名 speaker 进行投票，形成最终的邀请名单
